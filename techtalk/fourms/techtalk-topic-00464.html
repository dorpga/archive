Booting virtual machines

by Vas709 » Fri Jun 20, 2014 8:39 am

Hello,

Instead of using virtual machine clients such as VMware, Virtual PC or Virtualbox, we can boot into the virtual machine using a .vhd file that can boot guest operating system in a real mode. Without partitioning physical hard disk we can boot virtual hard disk in a real mode. Creating virtual hard disk can be a safe way, but re-partitioning real hard disk can lose all data, and therefore it needs a backup.
