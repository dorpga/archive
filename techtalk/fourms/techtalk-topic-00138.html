Registry hack enables continued updates for Windows XP

by V0rTeX » Mon May 26, 2014 10:33 pm

Create a text file with a .reg extension and the contents below:

Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SYSTEM\WPA\PosReady]
"Installed"=dword:00000001

Then run it.
It will make Windows Update to think you're running Windows POSReady 2009 (XP-Based Operating system which is still supported by Microsoft). All the updates for that thing are 100% compatible with a regular XP SP3.
