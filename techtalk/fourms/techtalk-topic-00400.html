(Humor) CONSYS.FIG

by maxell » Tue Jun 10, 2014 4:57 am

CONSYS.FIG is a config file usually found in installations of MS-DOS. CONSYS.FIG has many seeds, some examples are HISYS.MEM and EXE386.EMM. It is not viewable in a text editor, and instead is only viewable using SPEAKER.DRV and a very well-timed CTRL-ALT-DEL-SUPER-META-COMMAND-SELECT-START-SHRUBBERY press. If the file has opened correctly the Teenage Mutant Ninja Turtles will come out of the nearest manhole. However the turtles will only be fluent in Afrikaans. If the file is not opened successfully then..well, you don't wanna know.

CONSYS.FIG is also a file on the Apple Newton MessagePad, however it seems to be corrupted as it includes no seeds and is coated in cookie. It is not known what Apple's original plans for this file's use is, although it may involve a plot to kill off all rival fruits via imitation.
