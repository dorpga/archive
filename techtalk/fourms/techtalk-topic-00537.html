This one, I have no idea why (it's a fake language BTW)

Uses python-like structuring, but uses "with" for parameters. Square brackets can be used for math on the spot. Conditionals on the spot can be done too, and use "its" for specifying variables.

Code: Select all
import stdio

function print with x:
    stdio.print with [x + "\n"]

class repeater:
    repeat with 20:
        do print with [if x == "Hello" its x]

new string x with "Hello"
new repeater myrepeater


Statements:

function - makes new function
new - makes object. data types are in fact classes.
class - makes new class
import - imports a library
do - does a function
repeat - repeats something

And there would also be while, for, and if statements (which would be the same as python, except maybe with "with" instead of parentheses).

There. I call it SPL (Super Programming Language)
