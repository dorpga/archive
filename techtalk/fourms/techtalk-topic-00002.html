http://client00.chat.mibbit.com/?channe ... .fr.irc.tl

From here on out, you must use an IRC client that is installed locally on your computer, as all internet IRC clients, can be used as a proxy.

hexchat: http://hexchat.github.io
pidgin: http://pidgin.im

These clients are recommended. And on Linux you can usually install pidgin or hexchat with your package manager.
Code: Select all
Ubuntu/Debian:
sudo apt-get install pidgin
or
sudo apt-get install hexchat

Fedora/Red Hat:
yum install pidgin (as root)
or
yum install hexchat (as root)

Arch:
pacman -S pidgin (root)
or
pacman -S hexchat (root)

Others:
Refer to your distributions docs.


You are required to register to chat to prevent people from impersonating you:
/ns register password email

To login you have to type the command bellow in the chat
/ns identify yourpassword

(`/ns` is a shortcut for messaging the user nickserv in most irc clients)
