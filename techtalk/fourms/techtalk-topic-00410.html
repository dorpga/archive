Why Windows 8 is hard to use

by Vas709 » Fri Jun 13, 2014 6:56 pm

Hello,

I heard that some people have negative reactions to Windows 8. Some people don't know how to use Windows 8.

There is a full-screen feature called Metro style apps, also known as the Modern UI. Instead of being in the desktop you are presented to the Start screen. You can access to any of Metro features such as Windows Store, weather, Bing, and other features. It also allows you to access to the Desktop feature. But in the desktop there's no start button from the taskbar, but instead of it includes a hotspot in the lower-left hand corner of the screen, where the Start screen tile pops-up, and the Start button among "the charms". That is, if there isn't a start button for Windows 8, some people would just get confused and they wouldn't know where to find it. Another thing is that when you are in the Metro screen there's no X (exit) button. Some people would just think that there is no way to escape (but there is). You have to grab it up on the top of the screen and then put it down to the bottom. And you are back to the Start screen.
