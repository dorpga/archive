Re: What are your favorite programming languages?

by rezedus » Sun May 25, 2014 3:07 pm

My favorite programming languages are:
1) Visual Basic .NET
It's easy to use, yet it has the power of the .NET runtime libraries. I prefer it for smaller, not serious projects.
2) C#
It has got syntax similar to the other C family languages. Again, it has got the great power of the .NET runtime libraries. I prefer it for the more serious projects.

Right now I am also trying to learn C++, as native is usually faster than just-in-time compiling into native (.NET, Java). On the other side, a good feature of runtimes with a JIT compiler in the back is that running the application (given it hasn't got any platform specific code) is a question of installing the official runtime in the case of Java, or the company-backed unofficial port in case of .NET (Mono).
I am not talking about .NET Native now, as it's still a beta and it's not for regular uses. However, I think it'll be a useful addon for those, who didn't like .NET because of the JIT, as they'll be able to program native applications as they could in C++, with forms designer and other stuff.
