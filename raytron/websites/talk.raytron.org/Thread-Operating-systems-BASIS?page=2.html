Here's the timeline that I always make for BASIS releases. This timeline is focused on BASIS Lite, which is the operating system's 2nd 32-bit OS after BASIS Slate. This release will also be a first, as it is the first release to come without a codename.

Here is the timeline of events:

End 2013: First mention of a 32-bit version of BASIS, probably based off BASIS 4
January 28, 2014: BASIS Lite "Minimal" announced
March 7, 2014 - BASIS Lite starts development
March 8, 2014 - BASIS Lite Build 1 built
March 8, 2014 - Development halts
March 9, 2014 - Development restarts, Lubuntu 13.10 new base.
March 11, 2014 - BASIS Lite Build 1 built.
March 13, 2014 - BASIS Lite Build 2 built and released.
March 16, 2014 - BASIS Lite Build 3 built and released.
April 4, 2014 - Development reset, Linux Lite 1.0.8 new base
April 5, 2014 - BASIS Lite Build A built and released
April 15, 2014 - BASIS Lite Build B built and released
April 29, 2014 - BASIS Lite Build C built and released
May 12, 2014 - BASIS Lite development canceled
May 22, 2014 - BASIS Legacy announced
July 22, 2014 - BASIS Lite announced
July 23, 2014 - Development starts, Xubuntu 14.04.1 LTS new base
July 31, 2014 - Development restarts, Xubuntu 12.04.5 LTS new base
August 20, 2014 - BASIS Lite Build 1 built and released.
September 6, 2014 - BASIS Lite build 2 built
September 7, 2014 - BASIS Lite build 2 released
September 20, 2014 - BASIS Lite build 3 built and released
September 29, 2014 - Shellshock vulnerability acknowledged and removed.
October 7, 2014 - BASIS Lite Alpha 1 built and released
October 11, 2014 - BASIS Lite Alpha 2 built
October 18, 2014 - BASIS Lite Beta 1 built
October 19, 2014 - BASIS Lite Beta 1 released
October 29, 2014 - BASIS Lite RC 1 built and released
October 30, 2014 - BASIS Lite Final built and released

This is the whole timeline for BASIS Lite. Thanks for reading.
