I would never run a VM on anything under 4GB if I were you but maybe try allocating less RAM to the VM itself. WinXP's default RAM usage in VMware may be more than half your system RAM!

Also, if you're using Linux you can pretty much just delete your swap partition from the fstab and the kernel will only use your faster system memory. Maybe your system is abusing the swap partition/file?
