Debian/SME Server Linux user

by kjell159 » Fri May 23, 2014 9:20 pm

Hello,
my name is Kjell and I live in Belgium, Europe.

I use a Debian linux derivative (called Tango Studio) on my laptop and I'm trying to setup a server on my older desktop computer (+6 years) with the Linux distro SME Server.
I have been using Linux for around 1 year and I probably never want to go back to Windows.
I'm into open-source software.
I really don't like Microsoft and Microsoft related products except for MS Office that I primarily use in school.
Although I find myself using libreoffice (an open-source alternative) a lot more.
Still I'm interested in other OS'es (although Linux is just a kernel we'll call it a OS just to keep things simple here) and that's why I'm here I guess.

Also I'm really into everything music.
I play bass, guitar, I own/use a keyboard, I have some little instruments like: a harmonica, a ukulele, recorder, panflute, ...
But I also like recording/mixing/mastering music.

I own a little studio with all the stuff: studio monitors (speakers), some hardware like a stereo equalizer, soundmeter, amplifiers, acoustic foam, ...
I run Linux of course in my studio aswell (I use my laptop for this purpose aswell.)
It is quite unusual, most people use Mac OS X or Windows 7 for audio purposes but I like the shorter list of choices for audio plugins and of course the openness, durability, safety, stability, custimizability, the list goes on of my Linux system in general.

In the end I do not do this stuff on a professional level so I can definitely get away with my Linux system for all my purposes.
It's been a really long way, I have spend LOADS of time learning Linux but I'm quite familiar with the command line nowadays and the fact that I'm setting up a server, is a great learning tool aswell.
If you can handle it than in my opinion (GNU)/Linux is the best OS from the 'three, Win, OS X, Lnx'.

If you are less computer savvy I would guess OS X but I have never used it.
I like Apple because they have awesome hardware (It just works, is quite, mostly beatiful, integrates pretty well, ...) and an OS that (Sorry, it's 'just' my opinion.) doesn't suck to much.

Don't get me wrong, my Linux system has failed on me a lot of times but it was in most cases my fault because I can't work with my computer.
And that is probably the main reason why I'm here,
I like to spend a little amount of time to keep on learning about computers and everything related to it.
Everybody can learn to click some buttons without knowing what lies behind it, who made it, what backdoors maybe in it, but when it crashes you're stuck.
A Linux user mostly knows how his system works and can modify it so it does what the user wants to make it do,
not the other hand: Microsoft deciding how it looks, behaves, ...

Although I believe MINIX is the most powerful OS so far I don't use it because there is like -almost none?- development/support for it.
The maybe 500 (ported) programs on the system and the fact that it doesn't have a real 64 bit version definitely hold me back.
I probably be a Linux user for quite a while from now although I want to keep on experimenting.

In the end Windows, OS X and GNU/Linux are all built on a monokernel and because of this reason never will be as 'awesome' as something like Minix.

That is my opninion of course, I don't really see anybody here using Minix?

I have an account on the OSTalk forums but I have never been active on it so nobody probably knows me until now.

And I love wearing hats.
(Yes, I keep an eye out for Fedora Linux, I like the project but I couldn't get it installed on my desktop. I had a lot of problems with my peripherals not working with Linux kernel 3.X... + I keep the OS that I have now on my laptop.)
