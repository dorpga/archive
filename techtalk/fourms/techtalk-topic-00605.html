How to add a Youtube-embedding bbcode on phpbb!

by Delracman » Tue Jul 15, 2014 6:56 pm

I just did this on my prophpbb forum. It actually is pretty easy. Not sure if this belongs here but if it needs to be moved then one of the moderators can move it.
Admins, feel free to use this is add the Youtube bbcode to TechTalk.

1) Okay, so first, you need to go to the ACP (Admin Control Panel), and click on the "Posting" Tab.
2) You should already be on the Custom bbcode page, but if not, try to find it.
3) Once you're there, find the button that says, "Add a new BBCode". Click it.
4) Then, it asks you for bbcode usage, HTML replacement, and Help line. Don't worry, I'll tell you what to put in there!
5) In the bbcode usage: put:
Code: Select all
[youtube]{SIMPLETEXT}[/youtube]

6) In the HTML replacement, put:
Code: Select all
<iframe width="640" height="360" src="//www.youtube.com/embed/{SIMPLETEXT}" frameborder="0" allowfullscreen></iframe><br /><small><a

You can put a > at the end of the a, but it just makes text you put after the bbcode when you're posting a non-clickable link.
7) In the help line, put whatever you would like. I put:
Code: Select all
Insert YouTube Video ID# – Example: [youtube]z7hhDINyBP0[/youtube]

8) Click the checkbox that allows the youtube bbcode button to show.
9) Click "submit."

There, you did it!

Note that you CANNOT use this for the URL, you must use the video ID. The video ID is the number after the https://www.youtube.com/watch?v=
For example, if my URL was https://www.youtube.com/watch?v=j1kVNo6ars8 (real video, LG G3 drop test if you want to check it out!), my ID would be j1kVNo6ars8. No two ID's are the same, so don't worry.

If there are 2 = in the URL link, use the first one immediately after the /watch?v.

I tried the URL method, but it did not work. However, this did.

Hope this helps! Again, admins if you want to add a Youtube-embedding tag, just use this.
