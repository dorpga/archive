There is a way to boot Windows 10 in virtual hard disk (VHD). Just be careful when you are installing an operating system in another hard drive because you would lose all data unless you know how to install it in VHD.

Please note that you will not boot Windows 10 in VHD unless you have Windows 7 Enterprise or Ultimate edition. For more information about booting VHD see on the Microsoft website.

1. Create a new VHD using VirtualBox.
2. Mount ISO file in a VHD.
3. Install Windows 10.
4. When the installation is finished, you do not need to install Guest Additions featured in VirtualBox.
5. Close VirtualBox.
6. Download and install EasyBCD.
7. Find the VHD file you have created and add a boot entry to the boot menu.
8. Close EasyBCD, restart your computer and then boot the VHD file.

You can now use Windows 10. That's how I did when I installed Windows 8.1 in VHD and booted into it. But from now on I installed Windows 8.1 on a physical hard disk and there was no VHD. In Windows XP I cannot boot the VHD unless I have Windows 7 or 8.
