WinGroove Software Synthesizer

by Polygon6789 » Sun Sep 28, 2014 10:03 pm

If you tried to play a MIDI file in Microsoft Virtual PC 2004 or 2007, you probably know that it sounds like the equivalent of a broken trumpet and a tin can. This particular piece of software fixes that. WinGroove installs as a MIDI driver to provide high-quality MIDI into Virtual PC 2004 or 2007. It is shareware, and you are supposed to pay the developer money for it, but the author is no longer living from sources I've heard from. But I've included a working software key to register the software. Note that this only works on Windows 3.x and Windows 9x.
