by Leo7Mario » Sat Aug 02, 2014 4:34 pm

This is why I did not upgrade to Mavericks...

1: App Nap will interfere with Virtualbox (Even though I am completely aware that you can turn it off)
2: It will remove java and flash while you install 10.9. (You have to re-download Flash and Java)

I think that without those Annoyances OSX 10.9 will be a great OS. But I don't want to go to a OS which removes a bunch of stuff that I already have.
Unlike Early Versions of Windows When you upgrade from Windows 3.1 to 95 you can always go back with a uninstaller. It's not the same with Mac. (Unless you use Time Machine Which I don't use.)
