by phillips1012 » Thu May 29, 2014 3:27 pm

If you really do want to start off using arch linux, which I wouldn't really recommend if you have no linux experience, try manjaro. It has a graphical installer, is based on archlinux and comes with either xfce, openbox, kde, or none if you choose net edition. Manjaro net edition is pretty much user-friendly archlinux.

I would also recommend archbang, which has a graphical installer (well, it uses curses, a terminal library for making terminal apps that behave like graphical apps), but you have to manually partition your hard drive which can be confusing. It also does not come with pacman pre-configured so you have to configure it before you use pacman or else your system will explode.
