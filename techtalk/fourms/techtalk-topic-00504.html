Netscape is still alive as Seamonkey - and is a pretty good browser too! Seamonkey and Firefox share the "Gecko" engine, so you'll be getting the same rendering power as Firefox, but this time it comes with Email and IRC clients! It also has an HTML editor which I haven't used yet, but it is based off Netscape Composer so if any of know about that you'll be able to use it.

Guys, you should really download Seamonkey, it's my favourite web browser as I like it's layout. If you're nostalgic for the 2000s (I know I am) you can go View > Apply Theme > Seamonkey Modern, which will make it look like Netscape 7, when Seamonkey was forked from the original Navigator codebase (before Netscape became a crappy Firefox theme).
