by william.1008 » Fri Jun 06, 2014 8:37 pm

I have made a mumble sever for everyone to chat at. Mumble is like ICR but you can physically speak instead of typing.

If you use Windows or Mac/OS X you can download mumble for http://mumble.com

If you use a Linux based system type the install command into you terminal.

Fedora/Redhat (rpm):
Code: Select all
sudo yum install mumble


Ubuntu
Code: Select all
sudo apt-get install mumble


Arch/pacman based
Code: Select all
sudo pacman -S mumble


Please confirm that these work as I can only try the fedora one.

After installing and completing the setup, use the following setting to join:

Label: tech-Talk
Address: 216.119.158.164
Port: 3821
Username: Your Forum Name

The forum rules will apply while in the mumble server, there will also be one new rule:

No noises that other may consider rude, offensive, loud or annoying. Additionally, try to limit background noise.

I would like this to be made official, if the admins agree, could it please be moved to the announcements section, if not, pm me and I will label it as unofficial.
