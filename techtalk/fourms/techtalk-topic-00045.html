~~~*How to install IsaacLinux in virtual box*~~~~~~

1. Download the isaaclinux.vdi image
2. make a virtualbox session
3. IsaacLinux requirements:
a. 1GB to Up RAM
b. 10MB to up desktop fh
4. Now, go to your settings when your virtualbox session is selected.
5. Now go to storage.
6. Delete the S+ATA Controller.
7. Select the IDE controller above the SATA CONTROLLER.
8. Select the Harddisk Icon.
9. Choose, Choose Disk.
10. Locate the IsaacLinux.vdi and then click ok and close the settings then start the session ..
11. Have fun or make programs for isaacLinux.

~~*How to send your programs and patches for IsaacLinux.*~~
1. After you make the program, send you program to http://www.github.com/iansapelino/IsaacLinux
2. Read on the repository of Github in IsaacLinux.
3. You are now finish..
