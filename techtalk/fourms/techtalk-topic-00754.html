Microsoft Agent is a technology developed by Microsoft that can animate characters, move across the screen, and speak with a speech balloons. It was available for Windows 2000 and Windows XP, but not in Windows 7 because Microsoft is no longer supporting Microsoft Agent.

The video shows Merlin and James explaining how to use Microsoft Agent with Double Agent that can work on Windows 7 and later.

Yes, MASH can work. I installed James on the BellCraft technologies. And I installed CyberBuddy. CyberBuddy installed Merlin, Peedy, Genie and Robby on the characters folder. CyberBuddy is a program that uses Microsoft Agent. You can tell CyberBuddy to tell jokes, read the text, chat with people and so on. With normal Microsoft Agent, I can double-click James to stop talking. But with Double Agent, it won't stop him from talking. So I have to use DeskBot instead of CyberBuddy. I am still using Double Agent.
