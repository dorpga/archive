YugaNetwork Wrote:Talk about Windows 95 here
A significant upgrade over Windows 3.x.
Windows 95 has a significant upgrade in Usability. When it released it brought a clean, crisp UI that both ran fine on slower PCs and was zippy on fast PCs, networking was optional (IE was not even included in the original upgrade release on 95) and any accessory or Multimedia feature could be not installed during setup or removed during use with add/remove programs. It also added Win32c, which is the Win32 API finally embedded into Win386 (DOS based windows) and that was a near full implementation of the API, not just a subset.
Of course, it had it's problems, notably with stability (9x was kinda fussy) but most of the later versions of windows introduced problems of their own. (And yes, it had the con/con bug)

Windows 95 is..err...don't really have much to say about it. It's like giving a culinary review of corn flakes, it's just so plain you really can't say anything. Uh...it's windows...and......kinda.......grey.....'n' stuff

It's effectively the windowsiest windows version ever, if that makes any sense. It was adding crucial features to windows that by now, we just expect (cue Windows 8 circlejerk) and by being just the "essentials", even though they weren't essential when it came out, makes Windows 95 seem bland and boring.

Windows 95 not only defined the standard interface, but also just the aesthetic. If I say something like "It's weird how Windows 8 looks nothing like Windows" that really means "It's weird how Windows 8 doesn't have all the bevels and design cues of Windows 95". Windows 95 is "Windows" because the design hadn't changed for 6 years, and arguably longer. Windows could be configured to look like it up to Windows 7, which is still widely used. Server 2003 and 2008 still look like it by default. And the entire aesthetic of Windows XP basically amplified all the bevels, drop shadows, and ActiveIntelliPowerHyperness that came with Windows 95.

Windows 95 built the reputation of Windows as the most neutral, universal system - it's on the ATM. It's running TV stations. It's running on your school computers. It's running on your grandma's computer. It's running on your computer. Windows seems faceless. And the cruel irony of this is that most of these roles are being replaced by Linux (well, maybe not your grandma's computer just yet).

And Windows has a face. You know how I found out? Because I tried to find a face in Linux. You can't. Linux is a kernel, there's no frontend. Ubuntu has a face I guess, but the millions of servers running every day aren't powered by desktop Ubuntu. They're powered by Debian, or Red Hat, or SUSE, or maybe a server version of Ubuntu that looks nothing like a desktop. With that in mind, Linux to the non-tech-savvy or even to desktop power users is this vague...thing. There's a penguin, I guess, but what does that mean? Windows had a face of industry, of efficiency, of emotionlessness. Windows had a face of facelessness. Linux truly has no face.

Oh, Windows 95, yeah. Windows 95 is outdated for most tasks. You can't run Firefox on it. You can't get support for any sane graphics chipsets. Don't use it. Maybe in VMware or something.
