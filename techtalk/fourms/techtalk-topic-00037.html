Want to review a linux distro, post it here.

I have recently switched from Windows 8.1 to Fedora 20 and would like to give my review.

Fedora linux by default comes with Gnome 3, which is certainly not for those who like a traditional desktop. However, if, like like me, you came from from windows 8 and really like the start screen style interface, Gnome 3 is great. I like the start screen in that you can just start typing to get the app you want. However, it is so much more integrated with the rest of the OS and has many more features, like a side dock, dynamic multi-desktops and windows switcher.
Spoiler: show

The best thing about the the activities centre is the dymanic muilti-desktops. I have never used more then one desktop before and with other switching methods, I could never get used to them, but with this it just seem really natural.

But no distro is without faults, the thing I don't like about fedora is that is has no minimize or maximize buttons. windows are made full screen by dragging them to the top of the screen and snapping them. I'm sure there is something to fix this but it should be like this by default. Also, when system updates need installing there is not option to shutdown and install them, only restart. I only install update once I have finished what I'm doing, and when I'm finished I want to shutdown my computer, not restart it. Now it starts up again just too be shut down!Overall, I give Fedora 20 gnome version a 8/10
