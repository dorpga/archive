SUPER UPDATE! I have switched from Firefox to Qihoo 360 on Windows. (Note that this is a revision of my other update)
Reason for bumping: I don't want people to get the wrong information via this topic, also, this topic is not irrelevant because the subject isn't time-sensitive. (People will have favourite web browsers until web browsers become obsolete 100 years from now)

Qihoo 360:
Epicness! This browser is awesome, has a built-in ad blocker, and is based on Chromium. Plus it has an IE mode so you can make Nathan Lineback angry.

Chrome/Chromium:
This browser is super awesome because it's fast and has flash support, even on Linux. It's really modular as well. This is what I use on Linux and Mac OS X.

Firefox:
Same as chrome, except minus the flash.

Safari:
..good. This is the McDonalds of web browsers, at least to me. It's undeniably generic, and has some difficulties with things.

Classilla:

This web browser is based off of the ancient WaMCoM and shows. It's inabillity to render things properly has become somewhat of a--Classilla can't render this forum meme.

IE:

It's very much a meh browser like safari. The default metro interface personally drives me crazy, but there is a window option. The only reason this browser ever recieved so much hate was because of it's nasty business tactics in the '90s, and the fact that half of the Windows userbase can't get their head screwed on right and still use Windows XP + IE6. It's also not very secure at all, either.

Netscape:

Midori:
