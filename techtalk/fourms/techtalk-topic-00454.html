Linux Q&A incoming!

Q: What is Linux?
A: A unix-like kernel developed by Linus Torvalds. It's not an OS, that would be a linux distribution.

Q: What do Linux distributions/distros usually have?
A: They usually include a shell like bash or zsh, and some other GNU tools like gcc, cat, ls, and other things. Most include the X window system and some sort of desktop environment or window manager.

Q: What are some popular linux distributions?
A: Well Canonical's Ubuntu is probably the most popular due to it's focus on the web and it's included desktop environment, Unity, but other distributions like Fedora, Manjaro, Mageia, and BASIS are common.

Q: Where are my APPS!?!?!? WHAT DO THEY LOOK LIKE!?!?!?
A: Apps are usually stored in /bin and /usr/bin. They have no extension, and they have no need to. Just run an app by clicking on it in your file manager, or typing it's name and/or directory.

Q: Is linux text based?
A: Linux is text-based in the same way a burger is bun-based, you can just use a lettuce wrap and not have to worry about your heartburn or whatever. Linux has it's own built-in text buffer for apps that don't need or can't use the window system, like shells, old apps, rescue apps, or apps where it wouldn't be necessary.

Q: Is linux free?
A: Yes! Some of it's distributions include proprietary and sometimes costly software though, so certain distributions you'll have to pay for.

Q: Is linux for hipsters?
A: Considering hipsters are so ironic, they actually use Mac OS X.

You should add these Q&A's: (BTW, they're from my FAQ that KomputerKid refused to sticky.)

Spoiler: show
What is GNU/Linux?

The Linux kernel uses GNU tools to make it a full operating system. That's why some people call it GNU/Linux.

Can I get all the software that I use on Windows/Mac on Linux?

The answer is normally, but not always, no. You can get alternatives, though: you could use LibreOffice instead of MS Office, GIMP instead of Photoshop and so on. You could also use a program like Wine, that ports some Windows programs to Linux.

Is Linux easy to use?

Sometimes. Some Linux distributions are actually designed to be easy to use. Others, like Arch Linux, are designed for Linux geeks.

Should I switch to Linux?

If you want to, you can. There's a lot of guides out there in the internet on how to install Linux.

What is a Linux distribution?

A Linux distro is simply the Linux kernel with compiled packages, like the X window system.

What are some of the more common distros?

Ubuntu and Linux Mint are the most common.

How can I install Linux?

You can find a guide somewhere on the internet, but oh well, I'll post a simple one here.

1. Download UUI
2. Download the distro ISO file
3. Launch UUI
4. Follow the steps in UUI. (WARNING: This will wipe ALL your data on the selected drive. BE CAREFUL.)
5. Restart your computer
6. Stick the USB in.
7. Start your computer.

How much does Linux cost?

Nothing! It's totally free! For example, if you wanted to try Linux Mint, you would go to Mint's website and download the ISO file and use something like UUI to install it.

Can I try Linux without leaving Windows/Mac OS X?

Of course! You can either install it in a virtual machine, or dual-boot it. There's lots tutorials for both somewhere on the internet.

Can Linux only run on a PC?

No! Linux runs on mobile phones, servers, supercomputers, embedded systems like ATM's and more!



--Common Linux misconceptions--


Linux can't run any good software

Yes, it CAN. For example, do you want to create a document? Use OpenOffice or LibreOffice. Want to browse the web? Use Firefox or Chromium.

Linux is GNU based

No, Linux uses GNU software. Linux is not based off of anything. Linux is Unix-LIKE.

No one uses Linux

Actually, a LOT of people use Linux. 1 or 2% PC users, most touch-screen phones use Android, most servers use Linux, even supercomputers!
