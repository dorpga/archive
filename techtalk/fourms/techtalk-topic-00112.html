by MajorSky17 » Sun Jun 01, 2014 12:58 am

I'm just going to go with the most significant releases here...

1.x and 2.x = no real opinion as I have never really used these versions

3.1 = used very little and this was in the late 90s after I had already learnt to use Windows 98, therefore I found the UI a little confusing at first, but otherwise no major gripes

95 = very intuitive UI, good tutorial for first-time users

98 = an excellent development on 95, great customisability

ME = a good OS if multimedia is your thing more than running real-mode DOS apps, which by this stage (2000) was becoming less common and multimedia capabilities were indeed more in demand - whilst I like ME, I do feel that it suffered a bad reputation due to being rush-released and having a short shelf life

XP = great all-round OS and an excellent introduction to NT for consumers - intuitive UI, great Start menu improvements

Vista = needed more time spend to fine-tune it before general release, but I appreciate Microsoft wanted to get it out since it had to the public already seemed like they'd been working on it for about 6 years...very attractive UI and I love Aero and the DWM effects; excellent security improvements

7 = what Vista should have been; the new taskbar capabilities are simply wonderful

8 = too much change with too little help for consumers; woefully awful tutorial (if you can even call it that) and very badly-executed planning and marketing

8.1 = better than 8, but still not good enough; too little too late - I think Microsoft will have a hard time recovering from the 8.x era!

Windows 1.01 wasn't Microsoft's first OS, It's not even really an OS in itself. To run Windows 1.01-3.11 you needed to have a MS-DOS shell underneath it to run. The operating system of the computer was MS-DOS and Windows just provided a more visual interface for ease of use. Without MS-DOS, Windows would not run and you wouldn't be able to do anything with your system until you actually installed a OS.
Just because something is old doesn't mean it's useless, you just need to find more creative ways to use it!
