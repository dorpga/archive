Windows 10 - Development so far

by allstarsnorks2 » Sun Sep 21, 2014 5:35 am

Looks like the new builds of Windows 9 has been leaked. Today, I'll be talking about the next Windows, Windows 9. Build number in discussion is 9841.

There has been a previous post regarding Windows 9 leaks. You can view the topic here: viewtopic.php?f=35&amp;t=632. I'm making this as new features has been added since this post.

Start menu
Spoiler: show


The Start menu, removed in Windows 8 in favour of the Start screen, has been brought back in Windows 9. A hybrid of both the Start screen (Windows 8) and the Start menu (Windows 7), it has gotten mixed opinions ever since Build 2014. You can unpin the Live Tiles to make the Start menu look Windows 7-ish. The power buttons have been moved to the top, and the username has been moved from the top left side. The quick links to the right side of the Start menu in Windows 7 has not been brought back, unfortunately.


Multiple desktops
Spoiler: show


A feature already available in Linux and Mac OS X. Microsoft is now jumping on the bandwagon and is now offering multiple desktops. This new feature, for those not familiar with it, allows more programs to be open on different desktops. This feature can also clean the clutter on the desktops that Windows has been known for doing.


Apps

Applications will open in a window instead of taking up the full screen. This feature has been announced in the Build conference back in April. The windowed Modern apps will have an options button.


Improvements on File Explorer
Spoiler: show


A new "Home" location will give a peek of what files you've opened recently, favourite folders, and most recently used folders. This Home location is now the default view of File Explorer in Windows 9. Everything will stay the same, though.


Notification Centre
Spoiler: show


Notification Centre will be available on Windows 9. Like Mac OS X, it gives a peek of what the apps have pushed out. Not much has been said about this feature.


Cortana

It has been said that Cortana, the personal assistant in Windows Phone 8.1, will make its debut to desktops and tablets in Windows 9. Again, not much has been said to confirm the claim.

Microsoft will unveil the new operating system in Spetember 30. Be sure to check live blogs, as there will be no streams then.

What do you think? Do you like the new features?
