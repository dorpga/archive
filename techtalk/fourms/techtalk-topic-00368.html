Media Player (By Technology Area)

by Baltic » Thu Jun 05, 2014 4:12 am

I have been trying to make a media player of some sort for months now, but I always gave up, because I wanted, JUST a music player, but there was mostly tutorials for media players. Then I thought that if it had more functions, it could be better. So now that I have better coding skills, and know more about what I'm building, I have created Media Player (By Technology Area). It has a playlist type feature that lets you store music on there, than play it with starting the list. Also, something I almost threw out, a repeat check box, so every time it's checked, and at the end of a list, it will replay from the start. I'm working on making preset themes

NOTE: You must have version 0.1 installed before installing 0.2. Version 0.2 is set to be an update.
