My Wacky Idea

by Delracman » Fri Aug 22, 2014 1:44 am

Now there are offline apps for ChromeOS, but what if you could install anything? Run true desktop applications? Use a virtual machine and run Windows?
Well, with my wacky idea, you can. You will probably have to boot into developer mode on your Chromebook but once you do, get this program and then ChromeOS becomes a true Linux desktop!

Here's a screenshot. Note that the options listed are going to be for Installer, not for Emulator.

Spoiler: show


I also made a video I may post once I get closer to actually developing this program (if I do)

The boxes above the OS is a checkbox. The one below it is more options, where you select which version.

So basically, here are the options:
-Emulator, gets you access to the software store for that operating system (example being Ubuntu Software Center, Windows App Store, etc) and being able to download stuff from there to your computer.
-Virtual machine, pretty self-explanatory. It's like qemu where you can set which kind of processor you would like to use.
-Installer, download something from the internet and my magical program turns it into a file that ChromeOS can run.
-Modifier, make ChromeOS like another operating system, like skinning, and putting things on the desktop (this may not be possible, and if it is you probably need to be booted into developer mode and run some terminal commands)

What do you guys think? I'm not saying it's possible or I'm going to do it, just some thinking.
