Package Manager vs Software Center

by djdeedahx » Sun Sep 21, 2014 11:49 am

A package manager and a software center... Which do you prefer? Just to clarify, using the terminal is allowed as well.

I prefer Synaptic Package Manager! Not only it is MUCH LESS bloated than Ubuntu Software Center, which is total crap, it also lets you pick the packages you want and lets you know what other packages are needed to install the package you want.

I actually use this for Xubuntu 14.04.1 on my old laptop, I am not aware of anything else for other distros like Fedora or Arch Linux.
