XYZ Linux 0.0.2

by RainfallMuffin » Sun Jun 22, 2014 4:22 am

0.0.2.1 is here! However, ArchIn is in need of fixing. A remedy to this is to install the 'arch-install-scripts' package via pacman. You can load the live session with 'startxfce4'

XYZ Linux is a distro based off of Arch Linux. It is designed for ease of use and elegance. It uses the XFCE desktop environment with a top panel and a side panel. It also has a dock at the other side of the sidebar. Planned editions are XYZ/Solstice, interface designed off of a failed distro I once built, Has full and lite editions, (Lite edition included, but manual install required) and XYZ/Developer, A CLI with an optional GUI and a plethora of development apps.

(0.0.1 was released in February, but I took down the link)

Find it here: http://betaspace89.wordpress.com/xyz-linux/
