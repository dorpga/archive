1. Aero Glass is a transparent window which was made in Windows Vista and Windows 7.

2. Start Menu is where you can navigate and open program in the menu. It was made in Windows 95.

3. Legacy mode is a compatibility mode which can run older applications made for Windows XP, Windows 98 and earlier versions.

4. Classic theme is a skin which can be customized by changing colorful dialog box and other skins. It was made in Windows 95.
