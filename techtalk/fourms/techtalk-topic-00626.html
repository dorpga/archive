by phillips1012 » Sun Jul 20, 2014 10:50 am

I use FreeBSD, but I like all of the most common variations. Each is best used for a specific use:

  • FreeBSD - Good for desktop and server use. FreeBSD is the most multi purpose BSD, and has very scalable networking features. This one is well-rounded and excels at a variety of things. This is the debian of BSD.
  • PC-BSD - Good only for desktops and workstations. User friendly, but heavyweight. This is the k/x/lubuntu of BSD. (the k/x/lubuntu community is a lot different from the ubuntu community)
  • DragonflyBSD - Good for scientific computing and other systems that require high processing power. Not very useful to use on a desktop since the average desktop is not a supercomputer with 256 CPU cores.
  • NetBSD - Very lightweight. NetBSD can compile and run on almost every processor that exists. Good for embedded environments. Runs on toasters and potatoes. This is the Damn Small Linux of BSD.
  • OpenBSD - The OpenBSD team pays close attention to security and stability, making sure the code is perfect and vulnerability-free. Great for servers and some embedded environments. This is the CentOS of BSD.
  • Mac OS X - Proprietary and backed by a company people see as evil, but good for the average computer user, and friendly to developers. This OS unites unix, dumb users that rant about crap on facebook, and ruby programmers trying to get their massive Rails project to compile while using vim with 50 plugins for github and rails integration. This is the ubuntu of BSD.
  • ArchBSD - ArchBSD was a BSD for bleeding edge software, but is now a dead project

I know this topic is really old, and I apologize in advanced for posting but I wanted to put out my Favorite BSD...

I use FreeBSD as my daily driver, with Linux compatibility turned on for things like Flash and Skype. I use it with the i3 WM, and nano as my text editor for now. Eventually I will learn how vim works on FreeBSD, but vim on FreeBSD seems way more complex than vim on Linux.

I do like the other BSD distros though each have their own advantages:

FreeBSD: The all purpose flour of BSD. Can be used for any purpose, such as desktop, server, embedded system...
OpenBSD: The security based version of BSD. Best used on servers, supercomputers, and mainframes.
NetBSD: The DSL of BSD. Good for embedded systems. Will run on toasters, washing machines, potatoes, dryers, etc...
DragonFlyBSD: Good for server, supercomputer, and mainframe purposes.
PC-BSD: Good for the average desktop user (Not as good as OS X though). Easy to use, and good for somebody coming from Windows, or an easy Linux distro like Ubuntu.
Mac OS X: Yes it is BSD based. It is for the average home user, and has gotten more people using BSD, than ever before. Has significant market share (25%) and has brought UNIX to the home.
iOS: The most popular BSD distribution to date. Runs on apples iPhone/iPod/iPad. Has about 50-60% of the mobile market making it the most used BSD OS.
