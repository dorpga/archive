A dynamically typed language picks all the data types for you, so for example

x = 1 #Declare the variable x and assign it to 1. The compiler automatically knows it's an integer.
y = "The compiler knows this as a string"

As opposed to a statically typed language, which you have to tell what type a variable is.

int x = 1;
str y = "string";
