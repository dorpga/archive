Re: What are Statically Typed Languages

by Delracman » Sun May 25, 2014 12:28 am

In a statically typed language, every variable name is bound both a type and to an object.
In a dynamically typed language, every variable name is (unless it is null) bound only to an object.
In a weakly typed language, variables can be implicitly coerced to unrelated types, whereas in a strongly typed language they cannot, and an explicit conversion is required. In a typical weakly typed language, the number 9 and the string “9″ are interchangeable
