Mac OS X Yosemite

by muhuwj » Thu Jun 05, 2014 9:31 am

Mac OS X Yosemite is Apple's newest operating system. It is a huge change, and some of the biggest changes are listed here. The Notification Center now displays widget-like information, like you're calendar at a glance, and the weather. Spotlight can now search the App Store, Wikipedia, Maps, Bing, iTunes and iBooks besides you're calendar and files. iCloud Drive is a new iCloud feature that integrates iCloud with the desktop and the Finder. Safari seems more stripped-down than usual in this update, because in it's most basic view it shows you nothing except the webpage you want to view. Mail got a new Skitch-like feature called Markup that let's you edit pictures: write, draw, add text bubbles, etc. The new Handoff feature lets you start an activity on you're iPhone or iPad (like writing an email) and continue on you're Mac computer.
What are you're opinions on OS X Yosemite?
